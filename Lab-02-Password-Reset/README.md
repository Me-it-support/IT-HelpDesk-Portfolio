# Lab-02: Password Reset Issue

## Problem
A user (ramad) forgot their password and could not log into the system.

## Investigation
Opened Command Prompt as Administrator and checked existing users using:
net user

## Solution
Reset the user password using the following command:
net user ramad Ramadan123

## Result
The password was successfully changed, and the user can now log in.

## Tools Used
- Command Prompt (Admin)
- Windows User Management
