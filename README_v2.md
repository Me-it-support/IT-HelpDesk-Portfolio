# Lab-07: Shared Folder & Permissions

## Objective
Create a shared folder on Windows, configure permissions, simulate access issues, and resolve them to demonstrate IT Support L1 skills.

## Steps

1. **Create Folder**
   - Created a folder named `Lab-07-SharedFolder`
   - Shared it with the current user
   - Permission Level: Read/Write
   - Screenshot: `step-01-folder-created.png`

2. **Set Advanced Permissions**
   - Added another user
   - Configured Read, Write, and Full Control permissions
   - Screenshot: `step-02-permissions.png`

3. **Simulate Access Denied**
   - Logged in with the second user
   - Attempted to access the shared folder without proper permissions
   - Observed **Access Denied**
   - Screenshot: `step-03-access-denied.png`

4. **Fix Permissions**
   - Logged in as Administrator
   - Updated second user permissions (Write or Full Control)
   - Verified access from second user
   - Screenshot: `step-04-access-fixed.png`

## Notes
- Lab performed on a Windows machine.
- Demonstrates practical skills in File Sharing, Permissions, and troubleshooting Access Denied issues.
- No permanent changes made to the system.
