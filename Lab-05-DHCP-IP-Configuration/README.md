# Lab-05: DHCP & IP Configuration

## Objective
This lab demonstrates how to configure DHCP and IP addressing on a Windows machine using a Loopback Adapter.

## Tools Used
- Windows OS
- Loopback Adapter
- Command Prompt (ipconfig)

## Lab Steps
1. Install Loopback Adapter
2. Configure IP Address
3. Set up DHCP
4. Test IP assignment
5. Simulate IP conflict
