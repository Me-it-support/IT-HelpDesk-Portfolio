# Lab 08 - Basic LAN Network Setup

## 📌 Overview
This lab demonstrates the setup of a simple Local Area Network (LAN) using Cisco Packet Tracer. The network includes multiple PCs connected to a switch, which is further connected to a router.

---

## 🎯 Objectives
- Build a LAN topology with 4 PCs, a Switch, and a Router
- Configure IP addressing manually
- Set default gateway for all devices
- Verify network connectivity using ping tests

---

## 🧩 Network Topology
The network consists of:
- 1 Router
- 1 Switch
- 4 PCs

---

## ⚙️ IP Addressing Scheme

| Device | IP Address      | Subnet Mask       | Default Gateway |
|--------|----------------|------------------|-----------------|
| PC1    | 192.168.1.10   | 255.255.255.0    | 192.168.1.1     |
| PC2    | 192.168.1.11   | 255.255.255.0    | 192.168.1.1     |
| PC3    | 192.168.1.12   | 255.255.255.0    | 192.168.1.1     |
| PC4    | 192.168.1.13   | 255.255.255.0    | 192.168.1.1     |
