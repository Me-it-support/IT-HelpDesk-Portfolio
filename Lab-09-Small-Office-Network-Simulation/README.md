# 🏢 Lab 09 - Small Office Network Simulation

## 📌 Project Overview
This project simulates a small office network using Cisco Packet Tracer.  
It demonstrates how devices communicate in a real office environment including WiFi access, LAN connectivity, and network printing.

The scenario focuses on a simple office workflow where users (HR and staff) are connected through a switch and a wireless router, and a printer is shared across the network.

---

## 🖥️ Network Devices
- Cisco Router (1841)
- Switch
- Wireless Router (Access Point Mode)
- Laptop (HR Department)
- Laptop (Staff)
- Smartphone (WiFi Client)
- Network Printer

---

## 🌐 IP Addressing Scheme
- Network: 192.168.0.0/24
- Router Gateway: 192.168.0.1
- Wireless Router: 192.168.0.2
- Printer: 192.168.0.12
- HR Laptop: 192.168.0.10
- Staff Laptop: 192.168.0.11
- Smartphone: DHCP Assigned

---

## ⚙️ Configuration Highlights
- LAN setup using Switch
- Wireless network setup (SSID configured)
- DHCP enabled for automatic IP assignment
- Static IPs for key devices (Printer, Router, AP)
- Wireless and wired network integration
