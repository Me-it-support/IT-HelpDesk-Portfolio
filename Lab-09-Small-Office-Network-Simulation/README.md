# 🏢 Lab 09 - Small Office Network Simulation

## 📌 Overview
This project simulates a small office network using Cisco Packet Tracer, focusing on LAN, WiFi connectivity, DHCP, and basic device communication.

## 🖥️ Devices
- Cisco Router (1841)
- Switch
- Wireless Router (Access Point)
- HR Laptop
- Staff Laptop
- HR Mobile (WiFi Client)
- Network Printer

## 🌐 IP Scheme
- Network: 192.168.0.0/24
- Gateway: 192.168.0.1
- Printer: 192.168.0.12
- HR Laptop: 192.168.0.10
- Staff Laptop: 192.168.0.11
- Mobile: DHCP

## ⚙️ Configuration
- LAN setup via switch
- Wireless network with SSID
- DHCP for automatic IP assignment
- Static IPs for critical devices

## 🧪 Testing
- Ping between devices ✔️  
- Wireless connection ✔️  
- Printer connectivity ✔️
