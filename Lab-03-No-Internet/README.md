# Lab-03: No Internet Connection Issue

## Problem
User reported that their internet was not working.

## Investigation
- Checked IP configuration using 'ipconfig'
- Tested connectivity using 'ping 8.8.8.8'

## Result
- IP configuration was valid
- Ping test succeeded, indicating network connectivity is functional

## Solution
- Issue likely due to DNS or browser configuration (educational scenario)
- No hardware or cable issues detected

## Tools Used
- Command Prompt
- ipconfig, ping
