# Lab-04: Network & Performance Troubleshooting

## Objective
Diagnose and fix slow internet and heavy PC performance as a practical IT Support exercise.

---

## Steps & Screenshots

### Step 1: Ping Test
Test network connectivity using `ping google.com`.
- Screenshot: `01_ping_test.png`

### Step 2: IP Configuration
Check network settings with `ipconfig /all`.
- Screenshot: `02_ipconfig.png`

### Step 3: CPU Usage
Check Task Manager → Performance → CPU.
- Screenshot: `03_cpu_usage.png`

### Step 4: RAM Usage
Check Task Manager → Performance → Memory.
- Screenshot: `04_ram_usage.png`

### Step 5: Disk Usage
Check Task Manager → Performance → Disk.
- Screenshot: `05_disk_usage.png`

### Step 6: After Optimization
Close unnecessary programs and restart to improve performance.
- Screenshot: `06_after_optimization.png`

---

## Diagnosis & Solution
- Main cause of slow performance: RAM almost full (3.9 / 4 GB)  
- Disk usage: normal (0%)  
- Steps taken:
  1. Closed unnecessary programs  
  2. Restarted system  
- Result: Improved system performance and responsiveness
